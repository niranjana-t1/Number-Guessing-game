# Number Guessing Game

A simple Python number guessing game built as a beginner project while learning Python.

## Description

The program generates a random number between 1 and 30.
The user has to guess the number. After each guess, the program prints:

* "Try higher" if the guess is too low.
* "Try lower" if the guess is too high.
* "You guessed right!" when the correct number is found.

This game also keeps track of the number of attempts and allows the user to play again after winning.

## Features

* Random secret number generation
* Input range validation (1–30)
* Attempt counter
* Replay option
* Feedback after each guess

## Concepts Used

* Variables
* User input
* Conditional statements 
* Loops (while)
* Functions
* Return values (True and False)
* Python random module

## How to Run

1. Download or clone the repository.
2. Run the program:

python numberguessing.py
